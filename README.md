# Análise de Engajamento em Redes Sociais 📊

Dashboard interativo desenvolvido no Power BI para analisar padrões de engajamento em diferentes plataformas de redes sociais.

## Objetivo
Identificar quais fatores influenciam o engajamento nas redes sociais, respondendo perguntas como:
- Qual plataforma gera mais engajamento?
- Quais categorias de conteúdo performam melhor?
- Quais os melhores horários e dias para postar?

## Ferramentas utilizadas
- Power BI (visualização e DAX)
- CSV (base de dados do Kaggle)

## Principais insights
- **Instagram** lidera em engajamento médio entre as plataformas
- Conteúdos de **Food e Fitness** geram mais engajamento por categoria
- Picos de engajamento concentrados nos horários da manhã e noite

## Estrutura do projeto
- `social_midia.pbix` — arquivo do dashboard Power BI
- `social_media_engagement_dataset.csv` — base de dados utilizada
